# TradeMind

TradeMind is a FinTech analytics platform designed to automate post-trade analysis.

## Tech Stack
- Python 3.12
- Django 6.0
- PostgreSQL
- Docker & Docker Compose
- uv (dependency management)
- Ruff + Pre-commit

## Current Status
- Project foundation complete
- Dockerized Django with PostgreSQL
- Domain modeling in progress

## Next Steps
- Trade domain modeling
- FIFO P&L engine
- Async CSV ingestion
